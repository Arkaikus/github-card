# Examples

This file shows various examples of GitHub cards generated by the API.

## Basic Usage

### Default Theme

```
https://your-worker.workers.dev/octocat
```

Generates a card with:
- Width: 600px (default)
- Height: 400px (default)
- Theme: default (light theme)

### Dark Theme

```
https://your-worker.workers.dev/octocat?theme=dark
```

### Ocean Theme

```
https://your-worker.workers.dev/octocat?theme=ocean
```

## Custom Sizes

### Large Card

```
https://your-worker.workers.dev/octocat?width=800&height=600
```

### Wide Banner

```
https://your-worker.workers.dev/octocat?width=1200&height=400
```

### Small Card

```
https://your-worker.workers.dev/octocat?width=400&height=300
```

## Combining Parameters

### Dark Theme, Custom Size

```
https://your-worker.workers.dev/torvalds?theme=dark&width=700&height=500
```

### Ocean Theme, Large

```
https://your-worker.workers.dev/gaearon?theme=ocean&width=900&height=600
```

## Theme Comparison

### Light (Default)

```
https://your-worker.workers.dev/octocat?theme=default
```

- Background: White
- Text: Dark gray
- Accent: Blue

### Dark

```
https://your-worker.workers.dev/octocat?theme=dark
```

- Background: Dark gray/black
- Text: Light gray
- Accent: Light blue

### Ocean

```
https://your-worker.workers.dev/octocat?theme=ocean
```

- Background: Deep blue
- Text: Light blue/white
- Accent: Sky blue

## Card Content

Each card displays:

1. **Avatar** - User's GitHub profile picture
2. **Name** - Full name (if provided)
3. **Username** - @username
4. **Bio** - User's bio (if provided)
5. **Stats**:
   - Number of public repositories
   - Follower count
   - Following count

## Size Constraints

- Minimum: 200x200 pixels
- Maximum: 2000x2000 pixels
- Recommended: 600x400 to 800x600 pixels

## Caching

Cards are cached for 1 hour to improve performance and reduce API calls.

To get fresh data:
- Wait for cache to expire (1 hour)
- Or update the URL (add a query parameter like `?v=2`)
